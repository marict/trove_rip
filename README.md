# trove_rip
Python3.8 Script to extract images from The Trove directories.


## Setup
Install python3.8
then
```
pip install request
pip install bs4
pip install tqdm
```

## Usage
Pass in the directory with the images you want to extract. 
Example:
```
python trove_rip.py https://thetrove.net/Assets/Map%20Assets/Interior/Decoration/
```
